# IC - Professional Career Portal (React + Tailwind)

This is a Vite + React + Tailwind starter project generated from the user's HTML/CSS site.

How to run:
1. Install dependencies:
   npm install

2. Run development server:
   npm run dev

Notes:
- Tailwind is configured via tailwind.config.cjs and postcss.config.cjs.
- Main files are src/App.jsx and src/index.css (Tailwind + small custom CSS).
- Adjust components and copy original assets as needed.
